# fullbright-mod
A fullbright mod for TerrariaModder framework.

# Building from source
Refer to [the framework repository](https://github.com/Inidar1/terraria-modder) for more details.
1. Clone in the same folder as src/Core
```
git clone https://github.com/Mqrch/fullbright-mod.git
cd fullbright-mod
```
2. Build
```
dotnet build FullbrightMod.csproj
```
3. Output in the same folder as src/
